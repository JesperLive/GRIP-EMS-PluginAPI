# GRIP-EMS Plugin API Wiki

This wiki is the community side of the GRIP-EMS Plugin API. Anyone can edit it — add tips, list a plugin you've built, or write up something the reference doesn't cover.

The **official documentation** lives on the docs site, which is built from the repo and is the authoritative reference:

- Docs site: https://JesperLive.github.io/GRIP-EMS-PluginAPI
- Getting started, API reference, guides, and the event catalog all live there.

## What goes where

- **Docs site (repo `docs/`)** — the authoritative reference and guides. Changes go through the repo, so they're reviewed and versioned.
- **This wiki** — community knowledge: a [plugin showcase](Plugin-Showcase), a community [FAQ](FAQ), tips, and patterns. Lower friction, anyone can edit.

## Wiki pages

- [FAQ](FAQ) — community questions and answers.
- [Plugin Showcase](Plugin-Showcase) — plugins built on the API.
- [Contributing](Contributing) — how to help with the wiki and the docs.

If you find something in the wiki that contradicts the docs site, the docs site wins — and please fix the wiki page or flag it.
